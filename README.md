# homebrew-star

## 🏃 Get Started

```shell
/bin/zsh -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```

## 🍺 User Guide

### Tap

```shell
brew tap Ssttar/star
```

### Install

```shell
brew install <cask/formula>
```

### Uninstall

```shell
brew uninstall <cask/formula>
```
